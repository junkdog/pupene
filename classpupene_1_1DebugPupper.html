Detailed Description

template<typename Wrapped>
class pupene::DebugPupper< Wrapped >

This pupper wraps another pupper, logging each call to Wrapped::begin(), Wrapped::pup() and wrapped::end().

If the wrapped pupper returns PupPolicy::consume_object for any object/member-of-object, the associated pup() functions will not be invoked.

Example

using pupene;
using int_string_pair = std::pair<uint32_t, std::string>;
std::vector<int_string_pair> pairs =
{{1, "one"}, {2, "two"}, {4, "four"}};
// alternative 1 - pups to std::cout
auto pupper = debug_pupper();
pup(pupper, pairs, {});
// alternative 2 - save output to string
std::string debug_str = to_debug(pairs);
